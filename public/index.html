YelpCamp

Yelp Camp is a personal project that allows you to view campgrounds with a short description. Once you signup you can begin to create your own campgrounds that includes a title, image address and short description. You will also be able to leave a comment or edit a campground of yours. It was build with NodeJS and Express on back end and has Passport authorization. It is connected to a non-sql MongoDB database.

Crwn Clothing

Crwn Clothing a is personal project to quickly demonstrate a fast and efficient e-commerce site build with React and Node.js and connected to a Mongo Database. You can browse and add items to your cart and I also integrated a payment system (Stripe) upon checkout. The site is also a PWA (Progressive Web App) meaning you can install it on your device and use it like a native app.
